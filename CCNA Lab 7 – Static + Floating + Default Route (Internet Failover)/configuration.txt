================================================
Router R1 Configuration
================================================

enable
configure terminal

hostname R1

interface g0/0
description LAN Network
ip address 192.168.1.1 255.255.255.0
no shutdown

interface g0/1
description Primary ISP Link
ip address 10.0.12.1 255.255.255.252
no shutdown

interface g0/2
description Backup ISP Link
ip address 10.0.13.1 255.255.255.252
no shutdown


------------------------------------------------
Default Route (Primary Internet - ISP1)
------------------------------------------------

ip route 0.0.0.0 0.0.0.0 10.0.12.2


------------------------------------------------
Floating Static Route (Backup Internet - ISP2)
------------------------------------------------

ip route 0.0.0.0 0.0.0.0 10.0.13.2 10

end
write memory


================================================
ISP1 Router Configuration (Primary ISP)
================================================

enable
configure terminal

hostname ISP1

interface g0/0
description Link to R1
ip address 10.0.12.2 255.255.255.252
no shutdown

interface g0/1
description Internet Network
ip address 8.8.8.1 255.255.255.0
no shutdown

ip route 192.168.1.0 255.255.255.0 10.0.12.1

end
write memory


================================================
ISP2 Router Configuration (Backup ISP)
================================================

enable
configure terminal

hostname ISP2

interface g0/0
description Link to R1
ip address 10.0.13.2 255.255.255.252
no shutdown

interface g0/1
description Internet Network
ip address 8.8.8.2 255.255.255.0
no shutdown

ip route 192.168.1.0 255.255.255.0 10.0.13.1

end
write memory

================================================
Failover Testing
================================================

Step 1:
Normal Condition → ISP1 Active

Step 2:
Shutdown ISP1 link

interface g0/1
shutdown

Step 3:
Routing table automatically switches to ISP2

show ip route

Result:
Backup route becomes active and internet remains reachable.
