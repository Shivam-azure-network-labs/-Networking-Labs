# Networking Labs

Hands-on Cisco Networking Labs built using Cisco Packet Tracer.  
This repository contains practical networking scenarios focused on routing, switching, redundancy, and troubleshooting.

---

## Skills Covered

- Static Routing
- Floating Static Route
- VLAN & Inter-VLAN Routing
- OSPF (Single Area & Multi Area)
- HSRP Redundancy
- Route Redistribution
- Network Troubleshooting

---

## Lab Environment

Tool Used:  
Cisco Packet Tracer

Each lab includes:

- Network Topology
- Device Configuration
- Concept Explanation

The goal of this repository is to build strong practical networking skills by implementing real-world network configurations and troubleshooting scenarios.
