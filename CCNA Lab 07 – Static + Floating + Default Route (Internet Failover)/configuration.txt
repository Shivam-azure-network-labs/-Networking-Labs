==============================
Router R1 Configuration
=======================

enable
configure terminal

hostname R1

interface g0/0
description LAN Network
ip address 192.168.1.1 255.255.255.0
no shutdown

interface g0/1
description Link to ISP1 (Primary)
ip address 10.0.12.1 255.255.255.252
no shutdown

interface g0/2
description Link to ISP2 (Backup)
ip address 10.0.13.1 255.255.255.252
no shutdown

==============================
Static Default Route (Primary)
==============================

ip route 0.0.0.0 0.0.0.0 10.0.12.2

==============================
Floating Static Route (Backup)
==============================

ip route 0.0.0.0 0.0.0.0 10.0.13.2 10

==============================
ISP1 Router Configuration
=========================

enable
configure terminal

hostname ISP1

interface g0/0
ip address 10.0.12.2 255.255.255.252
no shutdown

interface g0/1
ip address 8.8.8.1 255.255.255.0
no shutdown

ip route 192.168.1.0 255.255.255.0 10.0.12.1

end
write memory

==============================
ISP2 Router Configuration
=========================

enable
configure terminal

hostname ISP2

interface g0/0
ip address 10.0.13.2 255.255.255.252
no shutdown

interface g0/1
ip address 8.8.8.2 255.255.255.0
no shutdown

ip route 192.168.1.0 255.255.255.0 10.0.13.1

end
write memory

PC Configuration
================
